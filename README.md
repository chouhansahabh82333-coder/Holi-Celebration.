# 🎨 Holi: A Cinematic Experience

A premium digital celebration of the Indian Festival of Colors with professional themes and interactive splashes.

## 🚀 Quick Start

1. **Install Dependencies**: `npm install`
2. **Set API Key**: Create a `.env` file and add `GEMINI_API_KEY="your_key"`
3. **Run App**: `npm run dev`

## ✨ Key Features
- **Theme Switching**: Midnight, Royal Gold, and Emerald.
- **Interactive Splashes**: Click anywhere to paint the screen.
- **Bilingual**: English and Hindi support.
- **Community Greetings**: Post and view Holi wishes.
